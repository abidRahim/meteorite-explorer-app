# Meteorite explorer
---
Lets user search metoerite names, which returns a detailed tabular data of matching string.

***Running live on : https://mighty-meteorite.netlify.com***


## Run guide

`yarn build`

or 

`npm install`

## API used

```
https://data.nasa.gov/resource/gh4g-9sfh.json
```
